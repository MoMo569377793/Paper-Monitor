# 命令速查表

这是一页版速查表，只放最常用命令。

## 1. 先加载 API Key

```bash
source .codex.env
```

## 2. 首次初始化

抓两个领域各 100 篇：

```bash
python3 main.py --config config/config-poe.json fetch --source arxiv --source dblp --recent-limit 100 --page-size 50
```

首次全量逐篇总结，用 `Ikun` 深读：

```bash
python3 main.py --config config/config-ikun.json enrich --limit 200 --with-llm --workers 2
```

生成总览，领域概览由 `Poe` 写：

```bash
python3 main.py --config config/config-poe.json catalog-report --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

生成周报，领域总结由 `Poe` 写：

```bash
python3 main.py --config config/config-poe.json report --type weekly --days 7 --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

## 3. 日常增量

一键执行：抓取 + 双模型单篇总结 + 日报：

```bash
python3 main.py --config config/config-poe.json run-once --type daily --source arxiv --source dblp --page-size 20 --since-last-run --enrich --enrich-limit 20 --with-llm --extra-config config/config-ikun.json --workers 2 --digest-primary-only
```

拆开执行：

```bash
python3 main.py --config config/config-poe.json fetch --source arxiv --source dblp --page-size 20 --since-last-run
python3 main.py --config config/config-poe.json enrich --limit 20 --with-llm --extra-config config/config-ikun.json --workers 2 --since-last-run
python3 main.py --config config/config-poe.json report --type daily --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

## 4. 新增论文很多时，节省 Ikun 配额

```bash
python3 main.py --config config/config-poe.json enrich --limit 20 --with-llm --extra-config config/config-ikun.json --workers 2 --since-last-run --secondary-on-priority --secondary-top-per-topic 5 --secondary-min-score 26
```

## 5. 单篇论文操作

查 `paper_id`：

```bash
python3 main.py --config config/config-poe.json paper-find --title "Hexcute"
python3 main.py --config config/config-poe.json paper-find --doi "10.1000/manual-study"
python3 main.py --config config/config-poe.json paper-find --arxiv-id "2504.16214v3"
```

只让 `Ikun` 深读一篇：

```bash
python3 main.py --config config/config-ikun.json enrich --paper-id 123 --with-llm --workers 2 --force
```

让两个模型都分析一篇：

```bash
python3 main.py --config config/config-poe.json enrich --paper-id 123 --with-llm --extra-config config/config-ikun.json --workers 2 --force
```

导出单篇报告：

```bash
python3 main.py --config config/config-poe.json paper-report --paper-id 123 --extra-config config/config-ikun.json
```

## 6. 手动维护论文库

手动加论文：

```bash
python3 main.py --config config/config-poe.json paper-add \
  --topic ai_operator_acceleration \
  --title "A Manual Paper" \
  --primary-url "https://example.com/paper" \
  --pdf-url "https://example.com/paper.pdf"
```

删除论文：

```bash
python3 main.py --config config/config-poe.json paper-delete --paper-id 123
```

## 7. 常用报告命令

日报：

```bash
python3 main.py --config config/config-poe.json report --type daily --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

周报：

```bash
python3 main.py --config config/config-poe.json report --type weekly --days 7 --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

总览：

```bash
python3 main.py --config config/config-poe.json catalog-report --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

模型对比报告：

```bash
python3 main.py compare-report --left-config config/config-poe.json --right-config config/config-ikun.json --type weekly --days 7
```

## 8. 守护运行

```bash
python3 main.py --config config/config-poe.json daemon --type daily --enrich --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

只跑 1 轮测试：

```bash
python3 main.py --config config/config-poe.json daemon --type daily --loops 1 --enrich --with-llm --extra-config config/config-ikun.json --digest-primary-only
```

## 9. 直接和模型聊天

和 `Poe` 聊天：

```bash
python3 chat_with_config.py --config config/config-poe.json
```

和 `Ikun` 聊天：

```bash
python3 chat_with_config.py --config config/config-ikun.json
```

单次提问：

```bash
python3 chat_with_config.py --config config/config-poe.json --message "请解释 Tensor Core 和普通 CUDA Core 的区别"
```

看实际 payload：

```bash
python3 chat_with_config.py --config config/config-poe.json --message "你好" --verbose
```

## 10. 最短上手路径

如果你现在就要开始，直接执行：

```bash
source .codex.env
python3 main.py --config config/config-poe.json fetch --source arxiv --source dblp --recent-limit 100 --page-size 50
python3 main.py --config config/config-ikun.json enrich --limit 200 --with-llm --workers 2
python3 main.py --config config/config-poe.json catalog-report --with-llm --extra-config config/config-ikun.json --digest-primary-only
```
